# Readory

Readory is a modern, feature-rich eBook reader built with Flutter. It supports PDF and EPUB formats, library management, collections, and progress tracking.

## Features

-   **Library Management**: Import books from device folders.
-   **Collections**: Organize books into custom collections (e.g., Favorites, Sci-Fi).
-   **Advanced Reader**:
    -   PDF & EPUB support.
    -   Table of Contents navigation.
    -   Thumbnails and easy page navigation.
    -   Bookmarks.
    -   Dark/Light mode themes.
-   **Reading Progress**: Automatically tracks where you left off.
-   **"Want to Read" & "Have Read"**: Status tracking for your library.

## Getting Started

### Prerequisites

-   Flutter SDK (3.0.0 or later)
-   Dart SDK

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/readory.git
    cd readory
    ```

2.  **Install dependencies**:
    ```bash
    flutter pub get
    ```

3.  **Run the app**:
    ```bash
    flutter run
    ```
